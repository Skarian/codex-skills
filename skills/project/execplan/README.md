# execplan

Create detailed plans for long-running agent work

Use this when a task is large enough that chat history is not a reliable source of truth: feature work, refactors, migrations, multi-step debugging, or anything that needs durable progress and validation notes

ExecPlans live under `.agent/execplans/`:

- `draft/`
- `active/`
- `archive/`
- `INDEX.md`

The workflow supports creating, executing, reviewing, and archiving plans
